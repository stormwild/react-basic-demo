# React Basics

## Setup Nodemon

```
npm i nodemon
```

## Setup Webpack

```
npm i -D webpack webpack-cli
npm i -D babel-loader
```

## Setup Babel for React

```
npm i @babel/core @babel/cli @babel/preset-env
npm i @babel/preset-react
```
